Background

Shy Kitty is like Hello Kitty, but doesn't say as much because she's shy. This site was conceived as a partially stylized version of the tutorial that I was working on at the time. It's name was inspired by the Heroku generated app name, which struck me as both very Japanese and with a subtle appeal. The original name was something like gentle-meadow889136. Since then, I've now gone through several web development, and returning the HTML lesson was a filling in of some of the missed details in doing the full rails tutorial.
